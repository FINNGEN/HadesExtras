# Cohort Short Name Utilities

Function to generate concise, human-readable short names for cohorts,
and to resolve conflicts to ensure uniqueness of cohort short names.

## Usage

``` r
makeShortName(name, id)
```

## Arguments

- name:

  Character. The full name of the cohort.

- id:

  Integer or character. The id of the cohort.

## Value

`makeShortName`: A character value representing the short cohort name.
