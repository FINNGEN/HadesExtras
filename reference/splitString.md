# Split String into Valid Tokens

Splits a string into valid tokens based on certain rules and operators.
Valid tokens include "Upd", "Ip", "Mp", numeric values, and parentheses.
This function is useful for parsing strings containing a cohort
operation.

## Usage

``` r
splitString(string)
```

## Arguments

- string:

  The input string to be split into tokens.

## Value

A character vector containing valid tokens extracted from the input
string.
