# Convert Operation String to Binary Tree Representation

Converts an operation string into a binary tree representation, where
valid operators include "Upd", "Ip", and "Mp". This function is useful
for parsing operation strings and constructing binary trees for
evaluation.

## Usage

``` r
.operationStringToBinaryTree(operationString)
```

## Arguments

- operationString:

  The input string representing an operation to be converted into a
  binary tree.

## Value

A list representing the binary tree structure of the operation string.
