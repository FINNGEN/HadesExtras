# Set NA to 0 in a cohortsSummary, and tibles to be empty

Set NA to 0 in a cohortsSummary, and tibles to be empty

## Usage

``` r
correctEmptyCohortsInCohortsSummary(cohortsSummary)
```

## Arguments

- cohortsSummary:

  tibble in cohortsSummary format

## Value

A tibble with cohort summary structure
